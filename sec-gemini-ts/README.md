# Sec-Gemini TypeScript SDK

## Developement

### Install dependencies

```bash
npm install
```

### Using the local package

Build and then use npm link to use the package from source

```bash
npm run build
npm link
```

in your project/app

```bash
npm link p9sdk
```

### Build for release

```bash
npm run build
```

### Runing tests

```bash
npm run test
```

### Running demo

```bash
npm run demo
```
